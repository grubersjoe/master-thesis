Die vorliegende Masterarbeit beschäftigt sich mit Transpilierung statischer Typsysteme für JavaScript. Dabei werden zwei derzeit populäre Systeme betrachtet:
% einerseits Flow von Facebook und andererseits die von Microsoft kreierte Programmiersprache TypeScript. Ziel der Arbeit ist die Entwicklung eines Transpilers, der den gesamten Quelltext eines JavaScript-Projekts von Flow in äquivalentes TypeScript übersetzt. Eine derartige Migration wird innerhalb des Unternehmens angestrebt, da vermutet wird, dass TypeScript Flow hinsichtlich Performance, Typinferenz und Unterstützung externer Software-Bibliotheken überlegen ist. Die Verifizierung dieser Hypothesen auf Grundlage empirischer Daten ist Gegenstand der anschließenden Untersuchung der auf diese Weise migrierten Projekte.

\end{abstract}
